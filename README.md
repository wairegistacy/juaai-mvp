# JuaAI MVP

JuaAI is an AI-powered financial assistant for informal businesses.

It converts natural language transactions like:

"Sold 5 sodas at 50"

into structured financial data, business insights, and credit readiness.

## Features

- AI transaction extraction
- WhatsApp-style interface
- Business dashboard
- Financial readiness score
- Voice transaction input
- Supabase database storage

## Tech Stack

- FastAPI
- Groq LLM
- Supabase
- HTML/CSS dashboard
- Python 3.13

## Vision

To build the AI operating system for Africa’s informal economy.
